# Управление

**ЛКМ** - Выстрел
**ESC** - Пауза / Выход
**ENTER** - Начать игру в главном меню

# Уровни

 - **В игре предусмотрено 5 уровней:**
 - каждый уровень имеет время и цель (кол-во очков);
 - при успешном прохождении — переход к следующему уровню;
 - после 5 уровня появляется экран Victory;
 - если время вышло, а цель не достигнута — Game Over.

# Настройки

Все параметры можно менять в settings.py:
 - размер окна
 - скорость цепи
 - радиус шариков
 - расстояние между шариками
 - количество уровней
 - цвета
 - таймер
 - очки за шарики
 - и многое другое
